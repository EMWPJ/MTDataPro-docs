# 快速入门

本教程将引导您在 10 分钟内完成第一次 MT 数据处理。

## 基本流程

MTDataPro 的数据处理流程如下：

```mermaid
graph LR
    A[导入数据] --> B[查看时间序列]
    B --> C[FFT 处理]
    C --> D[数据筛选]
    D --> E[阻抗估计]
    E --> F[导出 EDI]
```

## 步骤 1：创建工程

1. 启动 MTDataPro
2. 点击 **文件 → 新建工程**
3. 输入工程名称，选择保存位置
4. 点击"创建"

## 步骤 2：导入数据

1. 在工程面板中右键点击"测点"
2. 选择"导入数据"
3. 选择仪器类型（如 Phoenix）
4. 浏览并选择数据文件
5. 点击"打开"

MTDataPro 将自动识别采样率、时间范围和通道配置。

## 步骤 3：查看时间序列

1. 双击工程面板中的测点
2. 时间序列视图将显示原始数据波形
3. 使用工具栏按钮进行缩放、平移
4. 右键菜单可切换通道显示

## 步骤 4：执行 FFT 处理

1. 选择要处理的时间序列
2. 点击 **处理 → 新建处理方案**
3. 配置处理参数：
   - 窗口长度
   - 重叠率
   - 窗函数类型
4. 点击"运行"开始处理

## 步骤 5：数据筛选

1. 在傅里叶系数视图中查看处理结果
2. 使用筛选工具移除质量差的数据点
3. 可选：启用参考道处理
4. 可选：调整筛选阈值

## 步骤 6：导出结果

1. 选择要导出的测点或数据组
2. 点击 **文件 → 导出**
3. 选择导出格式（EDI、J-format 等）
4. 设置导出选项
5. 点击"导出"
