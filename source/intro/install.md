# 安装

MTDataPro 是免安装软件，无需运行安装程序。下载压缩包后解压即可使用。

## 下载软件

从以下渠道获取 MTDataPro：

- **官方发布**：联系开发团队获取最新版本
- **版本更新**：关注更新通知获取新版本

## 解压软件

1. 下载 MTDataPro 压缩包（ZIP格式）
2. 解压到目标目录（如 `D:\Software\MTDataPro` 或 `C:\MTDataPro`）
3. 确保目录路径不包含中文字符或特殊符号

## 目录结构

解压后的目录结构如下：

```
MTDataPro/
├── MTDPxxxxV.exe      # V版本主程序（便携版，无需硬件注册）
├── MTDPxxxx.exe       # 注册版本主程序（需硬件绑定）
├── *.dll              # 运行时库（Torch、MKL、TBB、FFTW等）
├── CalibrationFiles/  # 标定文件目录
├── Configurations/    # 配置文件目录
├── tmp/               # 临时文件目录
├── Manual/            # 说明文档（HTML和PDF格式）
├── protected/         # 保护目录（勿删除）
├── license.lic        # 公共许可文件
└── ReleaseInfo.txt    # 发布信息
```

### 目录说明

| 目录/文件 | 说明 |
|----------|------|
| `MTDPxxxxV.exe` | V版本（便携版），即插即用，无需注册 |
| `MTDPxxxx.exe` | 注册版本，需绑定电脑硬件 |
| `CalibrationFiles/` | 仪器标定文件 |
| `Configurations/` | 用户配置文件 |
| `tmp/` | 临时工作目录 |
| `Manual/` | 软件说明书（HTML/PDF） |
| `protected/` | 软件保护目录，勿修改或删除 |

## 软件启动

### V版本（便携版）

双击 `MTDPxxxxV.exe` 即可直接运行，无需注册授权。

### 注册版本

首次运行注册版本需要激活：

1. 双击 `MTDPxxxx.exe` 启动 MTDataPro
2. 在授权对话框中输入序列号
3. 点击"验证"完成注册

序列号与电脑硬件绑定，每个序列号只能在授权的电脑上使用。

## 首次配置

首次运行时建议设置以下路径：

- **工程目录**：存储工程文件的默认位置
- **数据目录**：原始数据文件的位置
- **临时目录**：处理过程中的临时文件（建议使用 `tmp/` 目录）

### 选择界面语言

MTDataPro 支持中英文界面，可在"设置"中切换。

## 外部依赖

MTDataPro 依赖以下外部库（已包含在压缩包中）：

| 依赖 | 用途 |
|------|------|
| Intel MKL | 数学核心库，用于 FFT 计算 |
| FFTW | 快速傅里叶变换库 |
| PyTorch | 深度学习运行时 |
| TBB | 线程构建块 |
| TeeChart | 图表绑制组件 |
| 仪器 DLL | 各仪器专用通信库 |

## 常见问题

### 无法启动

- 检查是否缺少运行库（VC++ Redistributable）
- 确保解压路径不包含中文字符
- 查看日志文件获取错误信息

### 注册失败

- 检查序列号是否正确输入
- 确认是在授权的电脑上使用
- 联系技术支持获取帮助

## 软件更新

1. 下载新版本压缩包
2. 关闭正在运行的 MTDataPro
3. 解压到原安装目录，覆盖旧文件
4. 原有的工程文件和授权通常可以继续使用
