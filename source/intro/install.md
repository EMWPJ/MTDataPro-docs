# 安装

本章介绍 MTDataPro 的安装方法、授权配置和首次运行设置。

## 下载软件

从以下渠道获取 MTDataPro 安装包：

- **官方发布**：联系开发团队获取最新版本
- **版本更新**：关注更新通知获取新版本

## 安装步骤

### 1. 运行安装程序

双击 `MTDP_Setup.exe` 启动安装向导。

### 2. 接受许可协议

阅读并接受软件许可协议。

### 3. 选择安装位置

默认安装路径为 `C:\Program Files\MTDataPro`，可自定义修改。

### 4. 选择组件

| 组件 | 说明 | 必需 |
|------|------|------|
| 主程序 | MTDP.exe 核心程序 | 是 |
| 帮助文档 | 用户手册和示例 | 否 |
| 示例数据 | 演示用数据文件 | 否 |

### 5. 完成安装

点击"安装"按钮，等待安装完成（约 2-3 分钟）。

## 首次运行

安装完成后，首次运行 MTDataPro 需要进行以下配置：

### 输入授权码

1. 启动 MTDataPro
2. 在授权对话框中输入序列号
3. 点击"验证"完成授权

### 配置数据路径

首次运行时建议设置以下路径：

- **工程目录**：存储工程文件的默认位置
- **数据目录**：原始数据文件的位置
- **临时目录**：处理过程中的临时文件

### 选择界面语言

MTDataPro 支持中英文界面，可在"设置"中切换。

## 外部依赖

MTDataPro 依赖以下外部库（已包含在安装包中）：

| 依赖 | 用途 |
|------|------|
| Intel MKL | 数学核心库，用于 FFT 计算 |
| FFTW | 快速傅里叶变换库 |
| TeeChart | 图表绑制组件 |
| 仪器 DLL | 各仪器专用通信库 |

## 卸载

通过 Windows"添加或删除程序"功能卸载 MTDataPro，或运行安装目录下的卸载程序。

## 常见安装问题

### 安装失败

- 确保有管理员权限
- 关闭杀毒软件后重试
- 检查磁盘空间是否充足

### 无法启动

- 确认已正确安装授权
- 检查是否缺少运行库（VC++ Redistributable）
- 查看日志文件获取错误信息

### 授权验证失败

- 检查序列号是否正确输入
- 确认网络连接正常（在线验证）
- 联系技术支持获取帮助
