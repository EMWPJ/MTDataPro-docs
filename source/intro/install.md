# Installation

## Download

Obtain the latest version of MTDataPro from the official release page or contact the development team for access.

## Installation Steps

1. **Run the Installer**: Double-click `MTDP_Setup.exe` to launch the installer.
2. **License Agreement**: Accept the license terms.
3. **Choose Install Location**: Select the installation directory (default: `C:\Program Files\MTDataPro`).
4. **Create Shortcuts**: Choose whether to create desktop and start menu shortcuts.
5. **Complete Installation**: Click Install to begin. The process typically takes 2-3 minutes.
6. **Launch**: After installation, MTDataPro can be launched from the Start menu or desktop shortcut.

## First Launch

On first launch, you will be prompted to:

1. **Enter License Key**: Input your license key provided with your purchase.
2. **Configure Data Paths**: Set the default paths for projects and data files.
3. **Select Language**: Choose between English and Chinese interface.

## External Dependencies

MTDataPro requires the following external libraries (included in the installation):

- **Intel MKL**: Mathematical kernel library for FFT calculations
- **FFTW**: Fast Fourier Transform library
- **Instrument DLLs**: Required DLLs for specific instrument communication

These are automatically installed and configured during setup.

## Uninstall

To uninstall MTDataPro, use the Windows "Add or Remove Programs" feature, or run the uninstaller from the installation directory.
