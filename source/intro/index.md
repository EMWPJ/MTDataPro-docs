# Introduction

## What is MTDataPro?

MTDataPro is a comprehensive desktop application for Magnetotelluric (MT) and electromagnetic geophysical data processing. It provides tools for time series analysis, forward modeling, and data visualization across multiple geophysical instrument formats.

## Key Features

- **Multi-format Data Import**: Support for Phoenix, Metronix, LEMI, ATTS, RMT, and more
- **Time Series Processing**: FFT, filtering, calibration, and spectral analysis
- **Forward Modeling**: 1D and 3D electromagnetic forward calculations
- **Data Visualization**: Interactive charts and geographic visualization
- **Processing Pipeline**: Modular, configurable processing schemas
- **EDI Export**: Standard EDI format output for transfer functions

## System Requirements

- **Operating System**: Windows 10/11 (64-bit)
- **Memory**: 16 GB RAM recommended (8 GB minimum)
- **Storage**: 2 GB free disk space
- **Display**: 1920x1080 or higher resolution
