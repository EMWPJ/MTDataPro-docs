# 简介

MTDataPro（Magnetotelluric Data Processing）是一款专业的 MT 数据处理软件，用于大地电磁测深数据的处理和分析。

## 什么是 MTDataPro？

MTDataPro 是由 王培杰 开发和维护的大地电磁数据处理软件。该软件提供了从原始时间序列数据导入、处理、分析到结果导出的完整工作流程，支持多种主流 MT 仪器数据格式。

## 主要特点

- **多仪器支持**：Phoenix、Metronix、LEMI、ATTS、RMT 等主流仪器格式
- **完整处理流程**：时间序列预处理、FFT 频谱分析、传递函数估计
- **智能数据筛选**：自动质量评估、Robust 估计、参考道处理
- **正演建模**：1D/2D/3D 大地电磁正演计算
- **可视化输出**：专业图表绘制、地理信息展示
- **标准格式导出**：EDI、J-format 等国际标准格式
