# Appendix

## Contributing

We welcome contributions to the MTDataPro documentation. Please see the GitHub repository for details.

### Reporting Issues

Found a typo or incorrect information? Please report it via the [Issue Tracker](https://github.com/EMWPJ/MTDataPro-docs/issues).

### Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Glossary

- **MT**: Magnetotelluric
- **FFT**: Fast Fourier Transform
- **EDI**: Electrical Data Interchange (standard MT format)
- **Transfer Function**: Mathematical relationship between EM fields
- **Tipper**: Vertical magnetic field transfer function
- **Impedance**: Complex relationship between E and H fields
