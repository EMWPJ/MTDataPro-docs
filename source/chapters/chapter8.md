# 🧮 第8章 正演建模与合成数据

本章介绍正演建模和合成数据生成功能。

---

## 7.1 📐 正演建模概述

MTDP支持多种正演建模工具，用于计算给定电阻率模型的MT响应。

### 7.1.1 支持的建模类型

| 类型 | 说明 |
|-----|------|
| ⚡ 电偶极子正演 | 有限长电偶极子源响应 |
| 📊 MT 2D正演 | 二维地电模型 |
| 📦 MT 3D正演 | 三维地电模型 |

---

## 7.2 ⚡ 1D正演建模

1D正演建模用于计算层状地电模型的电磁响应，包括MT（大地电磁）和CS（可控源）两种模式。

### 7.2.1 Forward1DForm界面

打开方式：选择菜单 `正演 → 电偶极子正演`

#### 选项卡结构

| 选项卡 | 功能 |
|-------|------|
| 频率 | 设置计算频率范围 |
| 高度 | 设置测点高度参数 |
| 电阻率 | 设置层状模型电阻率 |
| CS站点 | 可控源站点设置 |

#### 频率设置

在频率选项卡中可以设置：

- **频率范围**：设置起始频率和终止频率
- **对数间隔**：按对数均匀分布采样点
- **自定义频率**：手动输入特定频率值

#### 地电模型设置

在电阻率选项卡中定义层状模型：

| 参数 | 说明 |
|-----|------|
| 层数 | 地电模型的总层数 |
| 层厚度 | 每层的厚度（米） |
| 电阻率 | 每层的电阻率值（Ω·m） |

### 7.2.2 操作步骤

1️⃣ 选择菜单 `正演 → 电偶极子正演`

2️⃣ 在频率选项卡设置频率范围
   - 选择预设频率组或自定义频率
   - 设置频率数量

3️⃣ 在电阻率选项卡设置地电模型
   - 添加或删除地层
   - 输入每层厚度和电阻率

4️⃣ 在CS站点选项卡（可选）配置可控源参数
   - 设置源的位置和参数
   - 配置收发距

5️⃣ 点击计算按钮执行正演

6️⃣ 查看MT/CS曲线结果
   - 视电阻率曲线
   - 相位曲线

### 7.2.3 导出功能

计算完成后可以导出结果：

| 导出选项 | 说明 |
|---------|------|
| 导出当前站点 | 仅导出当前选中站点的计算结果 |
| 导出DE站点 | 导出所有双极源站点 |
| 导出全部站点 | 导出所有站点的完整结果 |

## 7.3 📦 3D正演建模

3D正演建模用于计算复杂三维地电结构的电磁响应，支持规则网格和自适应网格两种方式。

### 7.3.1 MT3DModelForm界面

打开方式：选择菜单 `正演 → MT 3D模型`

#### 编辑工具

| 工具 | 功能 |
|-----|------|
| 点选 | 选择单个网格点 |
| 矩形框选 | 选择矩形区域 |
| 多边形框选 | 选择不规则区域 |

#### 模型操作

**网格操作：**
- 添加X/Y/Z坐标网格
- 删除选中网格
- 编辑网格属性

**属性编辑：**
- 编辑电阻率值
- 批量赋值
- 区域电阻率设置

**测点管理：**
- 添加测点
- 删除测点
- 编辑测点位置

**源管理：**
- 添加源
- 删除源
- 配置源参数

#### 文件操作

| 操作 | 说明 |
|-----|------|
| 新建模型 | 创建空白3D模型 |
| 打开已有模型 | 加载保存的模型文件 |
| 保存模型 | 保存当前模型 |

### 7.3.2 创建新3D模型

点击"新建"按钮打开NewMT3DModelForm对话框。

#### 网格参数

| 参数 | 说明 |
|-----|------|
| Dx, Dy, Dz | X/Y/Z方向的网格间距（米） |
| X数量 | X方向的网格数 |
| Y数量 | Y方向的网格数 |
| Z数量 | Z方向的网格数 |
| X扩展 | X方向边界扩展层数 |
| Y扩展 | Y方向边界扩展层数 |
| Z扩展 | Z方向边界扩展层数 |

#### 物理参数

| 参数 | 说明 |
|-----|------|
| 空气电阻率 | 空气层的电阻率值（通常设为1e8 Ω·m或更高） |
| 海水电阻率 | 海水层的电阻率值（约0.3 Ω·m） |
| 测点边距 | 测点到模型边界的距离 |

#### 创建步骤

1️⃣ 设置网格间距和数量

2️⃣ 配置边界扩展参数

3️⃣ 设置空气和海水电阻率

4️⃣ 点击确定创建模型

### 7.3.3 四叉树模型

打开方式：选择菜单 `正演 → MT 2D四叉树正演`

四叉树模型采用自适应网格细分技术，能够根据地质体的复杂程度自动调整网格密度。

#### 特点

- **自适应细分**：根据电阻率变化自动加密网格
- **高效计算**：在保证精度的同时减少计算量
- **复杂地质**：适合处理复杂地质体边界
- **双模式支持**：同时支持MT和CS正演计算

#### QuadTreeModelForm编辑工具

| 工具 | 功能 |
|-----|------|
| 矩形工具 | 绘制矩形区域并设置电阻率 |
| 多边形工具 | 绘制不规则区域并设置电阻率 |
| 点工具 | 设置单点或小区域的电阻率 |

#### 操作步骤

1️⃣ 选择菜单 `正演 → MT 2D四叉树正演`

2️⃣ 点击"新建"创建模型
   - 设置初始网格参数
   - 定义模型范围

3️⃣ 使用编辑工具绘制地电结构
   - 选择矩形或多边形工具
   - 在视图区域绘制目标区域
   - 输入电阻率值

4️⃣ 设置背景电阻率值

5️⃣ 配置频率范围
   - 设置计算频率
   - 选择频率分布方式

6️⃣ 配置测点位置
   - 添加测点
   - 设置测点坐标

7️⃣ 执行正演计算
   - 点击计算按钮
   - 等待计算完成

8️⃣ 查看响应曲线
   - 视电阻率响应
   - 相位响应

## 7.4 📊 模型源编辑

ModelSourceEditForm用于编辑3D模型中的场源参数。

### 7.4.1 源参数

| 参数 | 说明 |
|-----|------|
| X | 源的X坐标（米） |
| Y | 源的Y坐标（米） |
| Z | 源的深度（米） |
| AZM | 方位角（度），源的走向方向 |
| DIP | 倾角（度），源的倾斜角度 |

### 7.4.2 操作说明

1️⃣ 在3D模型界面选择要编辑的源

2️⃣ 点击"编辑源"按钮

3️⃣ 在弹出的对话框中修改参数

4️⃣ 点击确定保存更改

## 7.5 🎯 正演测点合并

ForwardSiteMergeForm用于合并多个正演测点的计算结果。

### 7.5.1 功能说明

当需要将多个单点正演结果整合成完整的测线数据时，可使用此功能：

- 合并多个站点的响应数据
- 生成测线剖面数据
- 便于后续处理和解释

### 7.5.2 操作步骤

1️⃣ 选择菜单 `正演 → 测点合并`

2️⃣ 添加要合并的测点数据文件

3️⃣ 调整测点顺序

4️⃣ 设置输出参数

5️⃣ 执行合并

6️⃣ 保存合并结果

---

## 7.6 🎲 合成时间序列

### 7.6.1 功能说明

合成时间序列模块（SyntheticTimeSeriesForm）用于生成模拟的MT观测数据，支持：

- **算法测试**：验证数据处理算法的正确性
- **方法验证**：对比理论值与处理结果
- **教学演示**：展示MT数据处理流程

### 7.6.2 界面结构

| 区域 | 功能 |
|-----|------|
| 站点列表 | 管理正演站点 |
| 噪声列表 | 配置噪声参数 |
| 调制列表 | 设置幅度调制 |
| 方案列表 | 选择生成方案 |

### 7.6.3 噪声类型

支持10种噪声类型：

| 类型 | 说明 |
|-----|------|
| 矩形噪声 | 矩形脉冲噪声 |
| 阶跃噪声 | 阶跃函数噪声 |
| 三角噪声 | 三角波噪声 |
| 白噪声 | 均匀分布白噪声 |
| 高斯白噪声 | 正态分布噪声 |
| 指数噪声 | 指数分布噪声 |
| 瑞利噪声 | 瑞利分布噪声 |
| 厄兰噪声 | 厄兰分布噪声 |
| 椒盐噪声 | 脉冲噪声 |
| 余弦噪声 | 余弦调制噪声 |

### 7.6.4 噪声参数

| 参数 | 说明 |
|-----|------|
| Scale | 噪声幅度缩放 |
| TriggerFreq | 触发频率 |
| ContinueTime | 持续时间 |
| SampleRate | 采样率 |

### 7.6.5 幅度调制

**调制类型：**

| 类型 | 参数 |
|-----|------|
| 余弦函数 | 幅度、频率、相位 |
| 缩放函数 | 缩放因子A、B |

### 7.6.6 站点管理

**导入格式：**

- EM3DANI格式
- ModEM格式
- MT2D格式
- 线源格式

**导出格式：**

- EDI格式（张量、Y极化、X极化）
- 参数文件
- 坐标文件

### 7.6.7 生成参数

| 参数 | 说明 |
|-----|------|
| SampleRate | 采样频率（Hz） |
| MaxFrequency | 最大频率 |
| MinFrequency | 最小频率 |
| SamplingTime | 总采样时长 |

### 7.6.8 使用方法

1️⃣ 选择菜单 `正演 → 合成时间序列`

2️⃣ 加载或创建正演站点

3️⃣ 配置噪声参数（可选）

4️⃣ 设置幅度调制（可选）

5️⃣ 选择生成方案

6️⃣ 点击生成

7️⃣ 导出结果

## 7.7 💡 使用建议

### 7.7.1 模型建立

1️⃣ 从简单模型开始验证
   - 先用均匀半空间模型
   - 对比解析解验证程序正确性

2️⃣ 逐步增加复杂度
   - 添加层状结构
   - 引入横向变化

3️⃣ 与解析解对比验证
   - 1D模型可对比解析解
   - 检查数值计算精度

### 7.7.2 参数选择

| 参数 | 建议 |
|-----|------|
| 网格间距 | 目标体尺寸的1/3到1/5 |
| 边界扩展 | 至少5-10层网格 |
| 频率范围 | 根据探测深度要求选择 |
| 空气电阻率 | 建议1e8 Ω·m以上 |

### 7.7.3 计算效率

- 3D正演计算量较大，建议先做2D测试
- 利用对称性减少计算区域
- 合理选择网格密度
