# MTDataPro 软件中文手册

欢迎使用 **MTDataPro** —— 大地电磁（MT）和电磁地球物理数据处理软件。

本手册详尽介绍了 MTDataPro 的用法并提供了大量的实用示例，
既可以作为初学者的入门读物，也可以作为日常使用的参考书。

## 手册学习指南

本手册主要包含如下几个部分：

1. **入门指南**：介绍 MTDataPro 的安装方法、基本概念和快速入门教程。
2. **教程**：分模块详细介绍时间序列处理、正演建模、数据导入导出等功能。
3. **仪器支持**：Phoenix、Metronix、LEMI、ATTS、RMT 等主流仪器数据格式支持。
4. **实例展示**：丰富的实际数据处理案例和图件展示。
5. **附录**：常见问题、技术细节、贡献指南等。
