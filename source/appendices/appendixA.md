# 附录A: 快捷键参考

## 工程管理快捷键

| 快捷键 | 功能 |
|-------|------|
| Ctrl+N | 新建工程 |
| Ctrl+O | 打开工程 |
| Ctrl+S | 保存工程 |
| Ctrl+E | 导出数据 |
| Ctrl+I | 导入数据 |
| Ctrl+R | 重新打开最近工程 |

## 测点树导航

| 快捷键 | 功能 |
|-------|------|
| ↑ / ↓ | 在树形列表中上下移动 |
| Enter 或 空格 | 确认选择 |
| Delete | 删除选中项 |
| Backspace | 清空当前输入 |

## 图表操作

| 快捷键 | 功能 |
|-------|------|
| Ctrl + 拖动 | 平移图表 |
| Ctrl + Z | 撤销上一步选择（仅在图表选择模式下） |
| Ctrl + Y | 重做（仅在图表选择模式下） |
| ← / → | 在椭圆显示模式下切换椭圆选择 |

## 数据质量标记

| 快捷键 | 功能 |
|-------|------|
| Alt+1 | 标记质量等级1（优秀） |
| Alt+2 | 标记质量等级2（良好） |
| Alt+3 | 标记质量等级3（一般） |
| Alt+4 | 标记质量等级4（较差） |
| Alt+5 | 标记质量等级5（差） |
| Alt+A | 质量等级+1 |
| Alt+D | 质量等级-1 |
| Alt+↑ / Alt+↓ | 质量等级调整 |
| Alt+E | 设置EH类型 |

## 测点重排序

| 快捷键 | 功能 |
|-------|------|
| Ctrl+Shift+↑ | 将测点上移一位 |
| Ctrl+Shift+↓ | 将测点下移一位 |
