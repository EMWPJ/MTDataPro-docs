# 附录A: 快捷键参考

## 工程管理快捷键

| 快捷键 | 功能 |
|-------|------|
| Ctrl+N | 新建工程 |
| Ctrl+O | 打开工程 |
| Ctrl+S | 保存工程 |
| Ctrl+E | 导出数据 |
| Ctrl+I | 导入数据 |
| Ctrl+R | 重新打开最近工程 |

## 测点树导航

| 快捷键 | 功能 |
|-------|------|
| ↑ / ↓ | 在树形列表中上下移动 |
| Enter 或 空格 | 确认选择 |
| Delete | 删除选中项 |

## 图表操作

| 快捷键 | 功能 |
|-------|------|
| Ctrl（按下） | 开启/关闭图表平移模式 |
| Ctrl + Z | 撤销上一步选择（仅在图表选择模式下） |
| Ctrl + Y | 重做（仅在图表选择模式下） |
| ← / → | 在椭圆显示模式下切换椭圆选择 |
| Enter 或 空格 | 确认当前选择 |
| Backspace | 清除当前多边形选择（仅在多边形模式下） |
| Delete | 删除当前X波段范围 |
