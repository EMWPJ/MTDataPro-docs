# 附录A: 快捷键参考

## 常用快捷键

| 快捷键 | 功能 |
|-------|------|
| Ctrl+N | 新建工程 |
| Ctrl+O | 打开工程 |
| Ctrl+S | 保存工程 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Ctrl+A | 全选 |
| Ctrl+C | 复制 |
| Ctrl+V | 粘贴 |
| Ctrl+X | 剪切 |
| Delete | 删除选中项 |
| F5 | 刷新 |
| F11 | 全屏模式 |
| Ctrl+Q | 退出程序 |

---

## 处理相关快捷键

| 快捷键 | 功能 |
|-------|------|
| F9 | 执行FFT计算 |
| Ctrl+F9 | 批量FFT计算 |
| F10 | 数据筛选 |
| Ctrl+R | Robust估计 |

## 视图相关快捷键

| 快捷键 | 功能 |
|-------|------|
| Ctrl+1 | 时间序列视图 |
| Ctrl+2 | 傅里叶系数视图 |
| Ctrl+3 | 频谱图视图 |
| Ctrl+4 | 响应函数视图 |
| Ctrl+W | 关闭当前标签页 |
| Ctrl+Tab | 切换到下一个标签页 |
| Ctrl+Shift+Tab | 切换到上一个标签页 |
