# 附录F: 图表操作详解

本附录详细介绍MTDP中图表的各种操作方法。

---

## F.1 图表右键菜单

在任意图表上点击右键，弹出以下菜单：

### F.1.1 显示设置

| 菜单项 | 功能 |
|-------|------|
| 图表编辑器 | 打开完整的图表设置对话框 |
| 显示图例 | 显示/隐藏图例 |
| 显示命令栏 | 显示/隐藏图表工具栏 |
| 全屏显示 | 图表全屏模式 |

### F.1.2 坐标轴设置

| 菜单项 | 功能 |
|-------|------|
| 自动调整全部 | X、Y轴自动调整到最佳范围 |
| 自动调整X轴 | 仅X轴自动调整 |
| 自动调整Y轴 | 仅Y轴自动调整 |
| 恢复坐标轴 | 恢复到保存的坐标范围 |
| 记录坐标轴 | 保存当前坐标范围 |
| X轴对数 | X轴切换为对数坐标 |
| Y轴对数 | Y轴切换为对数坐标 |

### F.1.3 缩放功能

| 菜单项 | 功能 |
|-------|------|
| 放大X轴 | 放大X轴（缩小显示范围） |
| 放大Y轴 | 放大Y轴 |
| 缩小X轴 | 缩小X轴（增大显示范围） |
| 缩小Y轴 | 缩小Y轴 |

### F.1.4 导出功能

| 菜单项 | 格式 | 说明 |
|-------|------|------|
| 导出到剪贴板 | BMP | 复制到剪贴板 |
| 导出为PDF | PDF | 矢量格式，适合打印 |
| 导出为图片 | PNG/JPG | 位图格式 |
| 导出为SVG | SVG | 矢量格式 |
| 导出为HTML | HTML | 网页格式 |
| 导出数据 | TXT/CSV | 导出数据点 |

## F.2 坐标轴操作

### F.2.1 点击坐标轴设置范围

1. 点击图表的坐标轴（X轴或Y轴）
2. 在弹出对话框中设置：
   - 最小值
   - 最大值
   - 是否自动
3. 点击确定

### F.2.2 对数/线性坐标切换

- **X轴对数**：右键菜单 → X轴对数
- **Y轴对数**：右键菜单 → Y轴对数

> 💡 **提示**：视电阻率曲线通常使用双对数坐标。

## F.3 图表缩放与平移

### F.3.1 鼠标操作

| 操作 | 功能 |
|-----|------|
| 滚轮向上 | 放大 |
| 滚轮向下 | 缩小 |
| 左键拖动 | 平移图表 |
| 双击 | 恢复原始视图 |

### F.3.2 键盘快捷键（图表选择模式下）

| 快捷键 | 功能 |
|-------|------|
| Ctrl + 拖动 | 平移图表 |
| Ctrl + Z | 撤销上一步选择 |
| Ctrl + Y | 重做 |
| Enter 或 空格 | 确认当前选择 |
| Backspace | 清除当前选择区域 |
| ← / → | 在椭圆选择模式下切换椭圆 |

### F.3.3 缩放快捷键

| 快捷键 | 功能 |
|-------|------|
| Ctrl + + | 放大 |
| Ctrl + - | 缩小 |
| Ctrl + 0 | 恢复原始大小 |

## F.4 数据筛选图表操作

在数据筛选界面的图表支持多种选择方式：

### F.4.1 选择模式

通过下拉框或菜单切换选择模式：

| 模式 | 说明 | 操作方法 |
|-----|------|---------|
| 点选择 | 选择单个数据点 | 点击数据点 |
| 矩形选择 | 框选矩形区域 | 拖动绘制矩形 |
| 矩形反选 | 选择矩形外的点 | 拖动绘制矩形 |
| 多边形选择 | 选择多边形区域 | 依次点击顶点 |
| 多边形反选 | 选择多边形外的点 | 依次点击顶点 |
| X轴范围 | 选择X轴范围内的点 | 拖动设置范围 |
| Y轴范围 | 选择Y轴范围内的点 | 拖动设置范围 |
| X轴范围反选 | 选择X轴范围外的点 | 拖动设置范围 |
| Y轴范围反选 | 选择Y轴范围外的点 | 拖动设置范围 |

### F.4.2 完成选择

| 操作 | 功能 |
|-----|------|
| Enter | 完成当前选择 |
| 空格 | 完成当前选择 |
| Backspace | 清除当前选择区域 |

### F.4.3 撤销与重做

| 快捷键 | 功能 |
|-------|------|
| Ctrl + Z | 撤销上一步选择 |
| Ctrl + Y | 重做 |

### F.4.4 操作示例

**矩形选择：**
1. 选择"矩形选择"模式
2. 在图表上按住左键拖动
3. 松开鼠标完成矩形
4. 按 Enter 确认选择

**多边形选择：**
1. 选择"多边形选择"模式
2. 依次点击多边形顶点
3. 按 Enter 闭合多边形并确认

## F.5 添加临时曲线

可以在现有图表上叠加显示外部数据：

1. 右键菜单 → 添加临时曲线
2. 选择数据文件（TXT格式）
3. 数据格式：每行两个数值，空格或制表符分隔
4. 曲线自动添加到图表

**数据文件示例：**
```
0.01  100
0.02  95
0.05  90
0.1   85
```

## F.6 图表编辑器

右键菜单 → 图表编辑器，打开完整的图表设置对话框：

### F.6.1 常用设置

| 分类 | 可设置项 |
|-----|---------|
| 标题 | 标题文本、字体、颜色 |
| 坐标轴 | 范围、刻度、标签、网格 |
| 图例 | 位置、字体、可见性 |
| 系列 | 线型、颜色、标记 |
| 背景 | 颜色、渐变、边框 |

### F.6.2 保存设置

图表设置会自动保存，下次打开时恢复。

## F.7 多图表对比

### F.7.1 多测点叠加

1. 选择多个测点（Ctrl+点击）
2. 右键选择"对比曲线"
3. 所有曲线显示在同一图表

### F.7.2 图表工具栏

显示图表工具栏后可快速访问常用功能：
- 缩放工具
- 平移工具
- 复制
- 打印
- 保存

## F.8 常见问题

**Q: 如何恢复误操作的坐标范围？**

A: 右键菜单 → 恢复坐标轴，或双击图表恢复。

**Q: 如何同时调整多个图表的坐标？**

A: 目前需要分别调整每个图表。

**Q: 导出的图片模糊怎么办？**

A: 导出为PDF或SVG矢量格式，可无限放大。
