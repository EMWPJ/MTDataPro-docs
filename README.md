# MTDataPro Documentation

Chinese documentation site for MTDataPro software.

Built with [Sphinx](https://www.sphinx-doc.org/) and [myst-parser](https://myst-parser.readthedocs.io/).
Hosted on GitHub Pages.

## Quick Start

```bash
pip install -r requirements.txt
sphinx-build -b html source _build/html
```
